Classpublic class VizAlignGroup
InheritanceVizAlignGroup VizAlignTarget Object

The VizAlignGroup class represents a group of VizAlignTarget for alignment. using a group allows you to preserve relative positioning for multiple targets when doing alignment. this class acts much like its base but has one key difference, the updateGroupEnds(). this function is required so as to inform the targets of this group of the new alignment. this is the same functionality as using ctrl+g in the flash ide and doing alignment on the group
