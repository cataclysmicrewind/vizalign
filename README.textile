h1. VizAlign: h3. calm your ocd


VizAlign is a simple library that allows you to *vizually align* <code>DisplayObject</code>. A simple straightforward API with flexibility, boasting 40 alignment methods, grouping, and  makes VizAlign an amazing tool.

h3. super simple api:  
	
<code>VizAlign.align ([targets], [alignments], ignoreOrigins, applyResults, pixelHinting);</code>


h3. features:
	
 * simple api
 * 40 alignments
 * extendable framework
 * returns results for animation or storage
 * grouping!


h3. why

I made this library because of designers. My experience is that they are not even aware of the origin point of a Library asset (or DisplayObject), thus creating tons of hassle for everyone.

This library can ignore whatever nasty origin locations your designer throws at you.


h3. It's the Flash IDE align panel, but in code during runtime!, and way better ;)
